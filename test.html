üö® Common Issue: 403 Error with Ollama

If you get a 403 error with local LLM, Ollama needs CORS configuration:

Windows: $env:OLLAMA_ORIGINS="*" then ollama serve

Mac/Linux: export OLLAMA_ORIGINS="*" then ollama serve

This allows browser extensions to access the Ollama API.
